Code examples

Task:

Write a function camelize(str) that converts strings like my-short-string to myShortString — hyphens are removed, and all words after them get a capital letter.

Solution:

            
              function camelize(str) {
                let arrayStrings = str.split('-');
                
                let camelizedString = arrayStrings.map( (item, index) => {
                  if (index === 0) return item;
              
                  return item[0].toUpperCase() + item.slice(1);
                }).join('');
              
                return camelizedString;
              }
